Haekkhon PHP Framework

A modern, lightweight, and blazing fast PHP framework for building elegant applications

Get Started API Reference

Key Features

Simple Routing

Define routes with a simple function call. No complex configuration files or class hierarchies needed.

Dynamic Parameters

Support for dynamic route parameters using curly braces syntax like /api/users/{id}.

Base Path Support

Works seamlessly in subdirectories, perfect for shared hosting environments.

API & AJAX Ready

Automatic JSON responses for API routes and AJAX requests with proper content types.

Development Mode

Built-in debug information and detailed error messages in development environment.

Ultra Lightweight

Framework consists of just two core functions and a few lines of code. No dependencies.

Getting Started

1. Download the Framework

Download the haekkhon.php file and place it in your project directory.

2. Create Your Application

Create a new file (e.g., index.php) and include the framework:

index.php
// Include the Haekkhon framework require_once 'haekkhon.php'; // Define your routes route('/', function() { echo "<h1>Welcome to Haekkhon Framework!</h1>"; }); route('/hello', function() { echo "<h1>Hello, World!</h1>"; }); // Start the router dispatch();

3. Run Your Application

Access your application through a web server. The framework will automatically route requests to the appropriate handlers.

Code Examples

Basic Route

Define a simple route that responds with HTML:

basic-route.php
// Define a home route route('/', function() { echo "<h1>Welcome to our application!</h1>"; echo "<p>This is the home page.</p>"; });

Dynamic Routes

Create routes with dynamic parameters using curly braces:

dynamic-routes.php
// Route with single parameter route('/user/{id}', function($id) { echo "<h1>User ID: $id</h1>"; }); // Route with multiple parameters route('/api/posts/{category}/{id}', function($category, $id) { $data = [ 'category' => $category, 'id' => $id ]; // For API routes, return JSON header('Content-Type: application/json'); echo json_encode($data); });

RESTful API Example

Build a complete REST API with automatic JSON responses:

restful-api.php
// Sample data (in real app, this would come from database) $users = [ ['id' => '1', 'name' => 'John Doe', 'email' => '[email protected]'], ['id' => '2', 'name' => 'Jane Smith', 'email' => '[email protected]'] ]; // GET /api/users - Get all users route('/api/users', function() { global $users; header('Content-Type: application/json'); echo json_encode($users); }); // GET /api/users/{id} - Get specific user route('/api/users/{id}', function($id) { global $users; foreach ($users as $user) { if ($user['id'] == $id) { header('Content-Type: application/json'); echo json_encode($user); return; } } // User not found (returns 404 JSON automatically) });

API Reference

function route(string $uri, callable $handler)
Defines a route that maps a URI pattern to a callback function.
$uri string - The URL pattern to match (e.g., '/', '/hello', '/api/users/{id}')
$handler callable - The function to execute when this route is matched
function dispatch()
Processes the current request and routes it to the appropriate handler. Should be called after all routes are defined.
No parameters required

Route Parameters

Dynamic route parameters are defined using curly braces {parameterName}:

  • /user/{id} - Matches /user/123, /user/abc
  • /api/posts/{category}/{id} - Matches /api/posts/tech/456
  • Parameter values are passed to your handler function in the order they appear

Error Handling

The framework automatically handles 404 errors with appropriate responses:

  • API Routes: Returns JSON with application/json content type
  • Web Routes: Returns HTML response
  • AJAX Requests: Detected via X-Requested-With header
  • Development Mode: Includes debug information when APP_ENV=development

Complete Application Structure

Here's how to structure a complete application:

complete-app.php
require_once 'haekkhon.php'; // Set environment putenv('APP_ENV=development'); // Sample data $posts = [ ['id' => '1', 'title' => 'Getting Started with Haekkhon', 'content' => 'Learn the basics...'], ['id' => '2', 'title' => 'Building APIs', 'content' => 'Create RESTful APIs...'] ]; // Web routes route('/', function() { echo '<!DOCTYPE html> <html><head><title>My Blog</title></head> <body><h1>Welcome to My Blog</h1><p>Check out our <a href="/api/posts">API</a>.</p></body></html>'; }); route('/post/{id}', function($id) { global $posts; foreach ($posts as $post) { if ($post['id'] == $id) { echo "<h1>$post[title]</h1><p>$post[content]</p>"; return; } } echo "<h1>Post not found</h1>"; }); // API routes route('/api/posts', function() { global $posts; header('Content-Type: application/json'); echo json_encode($posts); }); route('/api/posts/{id}', function($id) { global $posts; foreach ($posts as $post) { if ($post['id'] == $id) { header('Content-Type: application/json'); echo json_encode($post); return; } } // 404 JSON will be returned automatically }); // Start the router dispatch();
